- a link to the repository or dataset landing page,

## Privacy and Responsible Use

The VEX dataset release is derived from an institutional short-answer grading context and is intended for research release and benchmark use. Users should not attempt re-identification of individuals or repurposing that conflicts with the anonymized and research-oriented nature of the release.

## Recommended Citation Practice

For the NeurIPS release, users should cite the associated VEX paper and reference the dataset version they used. If a formal citation file or paper DOI is added later, that citation should be preferred in downstream reuse.
