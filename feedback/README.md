# Feedback

This folder is reserved for feedback-specific analysis artifacts.

The dataset used for feedback experiments is documented under:

- [../dataset/additional/feedback_dataset/README.md](../dataset/additional/feedback_dataset/README.md)

The sampling procedure for the manual feedback-analysis subset is documented
under:

- [../annotation/feedback/README.md](../annotation/feedback/README.md)

At the current repository snapshot, this top-level folder does not contain the
main feedback dataset. It is kept as the coordination location for feedback
analysis outputs or paper-facing feedback artifacts that are not part of the
core dataset release.
